# Roadmap

## v0.1.0

- Local `.agent-memory/` storage.
- Agent-private notes.
- Evidence-gated shared memory.
- Markdown handoffs.
- `check`, `doctor`, and `status`.
- JSON schema export.
- `promote --from-note`.
- Agent integration templates.

## v0.2.0

- Configurable policy presets.
- GitHub Action for `agent-memory-isolation doctor`.
- Better stale-handoff detection.

## v0.3.0

- Optional DoneProof receipt linkage.
- Handoff compaction profiles for different agent tools.

## Future Ideas

- Optional SQLite backend.
- Optional vector index over promoted memory only.
- IDE snippets for Cursor and VS Code.
- Local dashboard for memory review.
- Multi-repo memory pack export.
